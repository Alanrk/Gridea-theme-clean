# Gridea-theme-clean
更新
2019.11.1:静态资源使用cdn加速

自制的一款免费的Gridea主题，在hexo同名主题上修改，包括首页、归档页、标签页等。
头部背景图在主题文件夹 media/images/home-bg.jpg，可自行替换，命名为 home-bg 即可。示例网站https://kveln.cn/
